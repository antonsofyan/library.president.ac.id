On Monday (13/03), Adam Kurniawan Library held an exhibition collaboration with Visual Communication Design Students, Interior Design Students, and Shutter Club. This event was held from 13 2023 until March 21, 2023. Various projects are displayed and can be enjoyed by students. These events were an implementation the function of library as a recreation place. Student artwork is expected to increase library visitors and spread new insights for civitas academics at President University (Anindya/Librarian).
